#  Support Ticket Classification & Prioritization

##  Project Overview
This project uses Natural Language Processing (NLP) to automatically classify customer support tickets and assign priority levels.


##  Objective
- Classify tickets into categories (Technical, Billing, etc.)
- Assign priority levels (Critical, High, Medium)


## Features
- Text preprocessing (cleaning, stopword removal)
- TF-IDF vectorization with n-grams
- Machine Learning classification (Naive Bayes)
- Hybrid priority system (ML + rule-based)


##  Results
- Category Classification: High accuracy
- Priority Prediction: Improved using hybrid approach


##  Techniques Used
- NLP (Text Cleaning)
- TF-IDF Vectorization
- Multinomial Naive Bayes


##  Business Impact
- Faster ticket routing
- Better prioritization
- Reduced response time
- Improved customer satisfaction


##  Dataset
Customer Support Ticket Dataset (Kaggle)
